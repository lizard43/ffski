# FFski
Fantasy Football Average Draft Position (ADP)

## Usage

- Install dependencies
    - npm install
- Start the expressjs REST API
    - npm run backend 
- Start the vue 3 frontend
    - npm run serve
- Open browser
    - [Local](http://localhost:8080/)
